# 用户指南

[English](user-guide.md)

## 概述

AI 计量经济学助手引导您完成结构化的计量研究工作流程：

1. **上传** — 导入数据集（CSV 或 Excel）
2. **画像** — 自动数据质量和结构分析
3. **规划** — 自然语言研究问题 → 推荐方法
4. **配置** — 选择变量、角色和转换
5. **运行** — 执行回归并输出完整诊断
6. **比较** — 多模型并排比较与推荐
7. **报告** — 生成可复现报告和出版级导出

---

## 项目

### 创建项目

1. 导航至 **项目** → **新建项目**
2. 填写标题、描述和研究问题
3. 添加标签以便组织（如 `panel-data`、`trade`）
4. 点击 **创建**

### 演示项目

在首页或项目页点击 **试用样本数据集**。这会创建一个预加载世界银行面板数据集和预填充研究问题的项目。

### 项目工作空间

每个项目包含：
- **概览** — 项目元数据、默认数据集、快捷链接
- **数据集** — 在项目内上传和管理数据集
- **时间线** — 所有项目活动的时间顺序日志
- **导出** — 出版级文档生成

## 数据集

### 支持格式

- CSV (`.csv`)
- Excel (`.xlsx`、`.xls`)
- 最大大小：50 MB

### 数据画像

上传后系统自动：
- 统计行数和列数
- 推断列类型（数值、分类、日期、标识符）
- 检测缺失值、异常值和偏度
- 识别面板、时间序列或横截面结构
- 建议合适的转换

### 结构检测

检测器识别：
- **面板数据** — 多个实体在多个时期的观测
- **时间序列** — 单个实体随时间变化
- **横截面** — 多个实体在某一时点

检测使用数据模式，不仅依赖列名。

---

## 研究规划

1. 导航至数据集的 **研究规划** 页面
2. 用自然语言输入研究问题（如"贸易开放度如何影响 GDP 增长？"）
3. 系统建议：
   - 因变量和自变量
   - 基于数据结构的推荐模型类型
   - 建议的转换
4. 审查并批准计划，或按需修改
5. 批准的计划可以指导建模步骤

**因果语言检测：** 如果问题使用因果语言（"导致"、"影响"、"造成"），系统会重新表述为关联性并添加明确警告。

## 变量配置

### 变量角色

| 角色 | 用途 |
|---|---|
| 因变量 | 结果变量 (Y) |
| 主解释变量 | 主要关注的解释变量 |
| 控制变量 | 减少遗漏变量偏误的附加回归量 |
| 实体 ID | 面板实体标识符（国家、企业等） |
| 时间 | 时间标识符（年份、季度等） |

### 预填充

系统根据列类型和数据结构预填充变量角色。您可以覆盖任何分配。

### 转换

可用转换：
- **对数** — 右偏变量（需正值）
- **缩尾** — 在指定百分位截断极端值
- **标准化** — 零均值、单位方差
- **插补** — 填充缺失值（均值、中位数或前向填充）
- **删除缺失** — 移除含缺失值的行
- **删除重复** — 移除重复行

## 模型执行

### 可用模型

| 模型 | 适用场景 |
|---|---|
| OLS | 横截面数据、基准规格 |
| 稳健 OLS (HC1) | Breusch-Pagan 指示异方差时 |
| 混合 OLS | 无实体效应的面板数据 |
| 固定效应 | 存在时间不变未观察异质性的面板数据 |
| 随机效应 | 实体效应与回归量不相关的面板数据 |
| 双向固定效应 | 同时含实体和时间效应的面板数据 |

### 诊断检验

每次分析包含：
- **VIF** — 方差膨胀因子检测多重共线性（警告 > 5，严重 > 10）
- **Breusch-Pagan** — 异方差检验
- **Jarque-Bera** — 残差正态性
- **Durbin-Watson** — 一阶自相关
- **Hausman 检验** — 固定效应 vs 随机效应选择（仅面板模型）

### 结果仪表板

运行分析后：
- 含标准误、t 统计量、p 值和显著性星号的系数表
- 系数图（森林图）
- 残差 vs 拟合值图
- 实际 vs 预测散点图
- 相关热力图
- 模型拟合指标（R²、调整 R²、F 统计量）

## 模型比较

1. 从任意数据集页面导航至 **比较模型**
2. 选择最多 6 个模型类型进行比较
3. 系统在相同变量上运行所有模型并展示：
   - 并排系数表
   - 拟合指标比较（R²、AIC、BIC）
   - 系数稳定性视图 — 主解释变量在不同规格中的表现
   - 多准则评分的透明推荐

### 推荐标准

| 准则 | 权重 |
|---|---|
| 结构兼容性 | 25% |
| Hausman 检验 | 20% |
| 异方差稳健性 | 15% |
| 模型拟合度 | 15% |
| 简约性 | 10% |
| 样本量 | 10% |
| 估计成功率 | 5% |

## 探索性发现

用于假设生成（非假设验证）：

1. 从数据集页面导航至 **探索关系**
2. 系统运行有界规格搜索：
   - 筛选潜在变量
   - 测试多种规格
   - 应用多重检验校正（Benjamini-Hochberg 或 Bonferroni）
   - 评估跨规格的稳定性
3. 发现按 6 个维度评分（校正支持、稳定性、方向一致性、数据质量、诊断、拟合度）
4. 一键将任何发现提升为理论驱动的研究计划

## 报告与导出

### 研究报告

从任意分析生成确定性叙述报告：
- 执行摘要
- 方法论部分
- 格式化系数表的结果
- 诊断解释
- 局限性与注意事项
- 支持 Markdown、HTML 和 JSON

### 出版导出

生成学术文档：
- **DOCX** — Word 文档，含封面、Times New Roman 字体、格式化表格、嵌入图表
- **LaTeX** — 独立 .tex 文件，含 tables/、figures/、appendix/ 目录
- **Markdown** — 带 *** 显著性标记的学术表格
- **JSON** — 程序化使用的结构化导出

每个导出包含：
- 出版级回归表
- 学术图表（系数图、残差图、相关热力图）
- 方法论附录
- 可复现附录（数据集校验和、软件版本、转换日志）

### 可复现性

每个分析档案包含：
- 完整的模型配置
- 数据集 SHA-256 校验和
- 含行数的转换日志
- 软件库版本
- 时间戳
