# 快速分析

[English version](quick-analyze.md)

快速分析是一个四阶段引导式工作流，无需统计学或计量经济学基础，即可从原始数据文件直接获得完整的计量分析结果。

---

## 何时使用快速分析

| 场景 | 建议 |
|------|------|
| 有 Excel 或 CSV 文件，希望立即获得结果 | 快速分析 |
| 有具体研究问题 | 快速分析（上传时输入研究问题） |
| 需要完全控制模型设定 | 高级工作流（项目） |
| 需要比较多个模型 | 高级工作流（项目） |

---

## 四个阶段

### 阶段一 — 上传

拖放文件，或点击 **选择文件** 打开文件选择器。

- 支持格式：`.csv`、`.xlsx`、`.xls`
- 建议最大文件大小：50 MB
- 文件保存在本地，不会发送至外部服务器

可以选填 **研究问题**（例如："GDP 增长是否影响失业率？"），帮助系统选择合适的模型。

点击 **上传并分析** 继续。

### 阶段二 — 规划（自动）

系统自动分析数据并生成分析计划：

- 检测面板数据、截面数据或时间序列结构
- 识别数值列和类别列
- 推荐分析模型（OLS、固定效应、随机效应或对数 OLS）
- 提议因变量和自变量组合

此阶段通常在数秒内自动完成。

### 阶段三 — 审核与确认

在运行分析前，先检查系统提议的方案：

| 字段 | 含义 |
|------|------|
| **推荐模型** | 系统自动选择的统计模型 |
| **因变量** | 需要解释的结果变量 |
| **自变量** | 用于解释结果的因素 |
| **分析意图** | 探索性分析或因果推断 |

您可以通过下拉菜单更改 **因变量** 和 **自变量**。如需更改推荐模型，请使用高级工作流。

点击 **确认并运行分析** 继续。

### 阶段四 — 结果

结果分三个部分展示：

#### 通俗解读

用非技术性语言解释分析结论，包括：
- 自变量整体上是否能解释因变量的变化
- 哪些具体变量具有统计显著性影响
- 任何数据质量警告（多重共线性、异方差、非正态）

#### 诊断统计表

| 诊断指标 | 参考阈值 | 含义 |
|----------|---------|------|
| R² | 越高越好 | 方差解释比例 |
| VIF（最大值） | < 10（≥ 5 时警告） | 多重共线性 |
| Breusch-Pagan p 值 | > 0.05 为佳 | 同方差性检验 |
| Jarque-Bera p 值 | > 0.05 为佳 | 残差正态性检验 |

#### 因果解释警告

若分析意图为 **探索性**，系统会显示黄色提示框，提醒相关关系不等于因果关系。因果推断需要适当的研究设计（自然实验、工具变量、双重差分等）。

---

## 结果页后续操作

在结果页可以：

- **导出结果** — 下载 Word、Markdown 或 PDF 报告
- **在项目中打开** — 在完整的项目工作流中进一步研究
- **重新开始** — 上传不同的文件

---

## 支持的模型

| 模型 | 选择条件 |
|------|---------|
| OLS（普通最小二乘） | 截面数据，连续因变量 |
| 对数 OLS | 截面数据，因变量分布偏斜（自动进行对数变换） |
| 固定效应（FE） | 面板数据，存在个体固定效应 |
| 随机效应（RE） | 面板数据，存在随机个体效应 |

模型根据数据结构自动选择。如需手动指定，请使用高级工作流。

## 常见问题

**上传失败，提示"不支持的文件类型"**
仅支持 `.csv`、`.xlsx` 和 `.xls` 格式。请将文件转换为其中一种格式后重试。

**规划阶段提示"无法检测数据结构"**
数据集至少需要两列数值型变量，且列标题必须在第一行。

**结果 R² 很低**
所选自变量对因变量的解释力较弱。建议在高级工作流中尝试不同的模型设定。

**出现"潜在多重共线性"警告**
两个或多个自变量高度相关，建议删除其中一个相关变量。

---

## 常见问答

**快速分析可以用于因果推断吗？**
快速分析提供的是基于相关关系的分析，并会在尝试因果解释时显示警告。真正的因果推断（双重差分、工具变量、断点回归）需使用高级工作流。

**我的数据会上传到云端吗？**
不会。所有数据均在您的计算机本地处理，不会发送至任何外部服务器。

**能否对同一文件重复运行快速分析？**
可以。每次上传都会创建新的会话，历史会话可在"项目"中查看。
